UDP Packet Forwarder

This section contains instructions for connecting to The Things Stack using the UDP Packet Forwarder.

In the Packet Forwarder menu, click the Plus button to create a new server.

In the server configuration options, check the Enabled box.

Choose Semtech as the Type.

For the Server Address choose custom, and enter the same as what you use instead of thethings.example.com in the Getting Started guide.

Choose the appropriate Port Up and Port Down values. These are both 1700 by default in The Things Stack.

Click Save to continue.

If your configuration was successful, your gateway will connect to The Things Stack after a couple of seconds.
