Configure V2 CLI

You will need to use the latest version of ttnctl, the CLI for The Things Network Stack V2 . Follow the instructions from The Things Network documentation. An overview is given below:

Download ttnctl for your operating system.

Update to the latest version:

$ ttnctl selfupdate

Go to https://account.thethingsnetwork.org and click ttnctl access code.

Use the returned code to login from the CLI with:

$ ttnctl user login "t9XPTwJl6shYSJSJxQ1QdATbs4u32D4Ib813-fO9Xlk"
